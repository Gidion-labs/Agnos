# Agnos

AI Dataset Preparation Platform — Web-Based

---

## About

Agnos is a web platform designed to help machine learning engineers **prepare training datasets efficiently**. It simplifies the most time-consuming parts of dataset preparation and allows users to focus on building AI models instead of manually cleaning data.

Agnos handles **large datasets**, performs deep quality checks, and delivers a **fast, responsive experience** with an **advanced dashboard and configurable settings** — no account required.

---

## Key Features

- Upload datasets in any format (CSV, JSON, TXT, etc.)  
- Clean and normalize data automatically  
- Tokenize text datasets for AI training  
- Chunk large datasets into manageable pieces  
- Remove duplicates and poor-quality entries  
- Advanced dashboard with customizable dataset preparation settings  
- Prepare **training-ready datasets instantly**  

---

## How It Works

1. Go to the website.  
2. Adjust the advanced settings to your needs.  
3. Upload your dataset.  
4. Preview results (optional).  
5. Download your cleaned and ready-to-use dataset.

> No coding or account setup required — just upload, configure, and process.
